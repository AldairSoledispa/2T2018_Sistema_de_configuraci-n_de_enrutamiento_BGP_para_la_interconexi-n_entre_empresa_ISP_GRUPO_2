Funcionamiento del programa en Python:
-Tener en cuenta la instalacion de un sistema gestor de base de datos.
  Para la correcta implementacion se utilizó el programa XAMPP para levantar los servicio de MYSQL y WEB para la creación de la base de datos utilizando el archivo "bgp.spl" dentro del directorio Codigo en este repositorio.
-El porgrama inicia al ejecutar la ventana principal denominada "main.c", misma donde se validará que los servicio del programa 
gestor se encuentren activos, caso contrario se lo indicará.
-Actualizar información en la base de datos a las condiciones particulares
  Esta implemenatción se llevó a cabo en equipos físico, por lo que las redes e interfaces presentes en la base de datos, para una validacion al iniciar el sistema, podrán traerle problemas ya que no coinciden con las que pueda implementar.
  
*****Una vez tenido en cuenta estas condiciones el programa funcionará correctamente.
